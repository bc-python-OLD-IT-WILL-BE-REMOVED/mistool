**Une belle nouveauté dans le module ``python_use`` :** la classe ``SKDict`` permet de définir des dictionnaires dont on peut extraire une valeur en utilisant une syntaxe du type ``myskdict["key_1", "subkey_1", "subsubkey_1"]`` au lieu de ``mymkdict["key_1"]["subkey_1"]["subsubkey_1"]``.

    SKDict --> Several Key Dict

    toc = SKDict({
        "I": {
            "A": {
                "1": "Dans I-A-1",
                "2": "Dans I-A-2",
            }
        }
    })


    print(toc)
    print()
    print(toc["I", "A", "2"])
